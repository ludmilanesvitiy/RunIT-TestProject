# RunIT TestProject 2019

## How to install

- `git clone`
- `npm i`

## Running end-to-end Protractor tests

- `npm run pre-protractor`
- `npm run protractor-e2e`

## Running end-to-end Cypress tests

- `npm run cypress-run`

- `npm run cypress-open`
